## 🚀 Getting Started

### Prerequisites
- Python 3.11+
- Node.js 20+
- Git
- Environment variables for integrations (see .env.example)

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/frankielee1971/soulcodes-unified.git
cd soulcodes-unified
```

2. **Set up backend environment**
```bash
cd backend
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your API keys and configuration
```

3. **Configure integrations**
```bash
# Add your API keys to .env:
# - NOTION_TOKEN
# - BREVO_API_KEY  
# - FACEBOOK_ACCESS_TOKEN
# - GOOGLE_SERVICE_ACCOUNT_FILE
```

4. **Start the backend server**
```bash
python main.py
```

5. **Open the frontend**
```bash
cd ../frontend
# Open enhanced_index.html in your browser
# Or serve with a local server for full functionality
```

### Agent TARS Setup

1. **Configure service accounts**
   - Set up Google Drive service account
   - Configure Notion integration
   - Set up Brevo API access
   - Configure Facebook Graph API

2. **Initialize automation**
```bash
cd agent
python agent_tars_enhanced.py
```

3. **Schedule automated tasks**
```bash
# Run the scheduler for continuous automation
python -c "from agent_tars_enhanced import AgentTARS; AgentTARS().run_scheduler()"
```

## 🔧 Configuration

### Environment Variables

Create a `.env` file in the backend directory with the following variables:

```env
# Notion Integration
NOTION_TOKEN=your_notion_integration_token
NOTION_TEMPLATES_DB_ID=your_templates_database_id
NOTION_ONBOARDING_DB_ID=your_onboarding_database_id
NOTION_BLUEPRINTS_DB_ID=your_blueprints_database_id

# Brevo Email Integration  
BREVO_API_KEY=your_brevo_api_key

# Facebook Group Integration
FACEBOOK_ACCESS_TOKEN=your_facebook_access_token
FACEBOOK_GROUP_ID=your_facebook_group_id

# Google Drive Integration
GOOGLE_SERVICE_ACCOUNT_FILE=path/to/service-account-key.json
GOOGLE_DRIVE_FOLDER_ID=your_google_drive_folder_id

# Backend Configuration
PORT=8000
DEBUG=True
CORS_ORIGINS=["http://localhost:3000", "https://soulcodes.site"]
```

### Archetype Configuration

Archetype definitions are stored in JSON files in the `archetypes/` directory. Each archetype includes:

- **archetypeName**: Display name
- **missionStatement**: Core purpose and calling
- **brandVoice**: Communication style
- **colorPalette**: Associated colors for theming
- **affirmations**: Power statements for daily activation

### Template Manifest

The `templates/template_manifest.json` file maps each archetype to:

- **notion_template**: URL to Notion template
- **email_sequence**: Brevo campaign identifier  
- **facebook_prompt**: Community activation prompt
